# 原材料出库

功能说明：
   用于根据领料单需求进行原材料的出库。
功能说明：
   查询待出库的原材料数据；
   点击目标钢板进行选料，推荐最佳钢板；
   可以切换其他库位查看其他满足条件的库位；
   点击出库进行选择预处理线库位/出库暂存区，原材料出库后进入到此库位；
对于非顶层原材料（板材类）进行出库，需要先进行移库处理。
