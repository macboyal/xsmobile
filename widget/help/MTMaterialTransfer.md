# 原材料移库
功能说明：
   用于现场对原材料库存进行移库操作，移库可以在库存总览中触发，也可以在出库过程中触发。
操作说明：
   支持选择原材料物资进行移库，选择移库目标库位进行确认，确认后 ，物资进入目标库位。
