# 半成品入库
功能说明：
   对半成品进行入库操作。
操作说明：
   【入库】：选择需要入库的数据后点击右下方的入库按钮，弹出选择库位的页面后选择一个库位，点击确认即可完成入库操作。入库完成后，数据进入半成品库存页面。
